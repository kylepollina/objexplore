# Object
An Interactive Python Object Explorer

![screenshot](images/screenshot.png)


## Install

```
pip install objexplore
```

or

```
pip install git+https://github.com/kylepollina/objexplore
```

## Usage

```python
from objexplore import explore
import pandas
explore(pandas)
```
